1. HTML

  • HTML - язык гипертекстовый разметки. Стандарты зашитые в браузер, благодаря которым он обрабатывает код и выдаёт нам страницу
  • Теги - блоки из которых состоит разметка (кирпичи сайта из которых он строится)
    1. Теги бывают:
      блочные(block) - занимают всю ширину страницы
      строчные(inline) - не больше ширины контента
    2. Всегда имеют общий вид <имя_тега>Содержимое тега</имя_тега>
    3. Внутри строчных тегов нельзя помещать блочные
    4. Атрибуты - это некоторые параметры html тегов, которые позволяют сделать их уникальными (атрибутов может быть у тего сколько угодно)
Какие бывают теги:
  1. <header></header> - шапка сайта в ней находится навигация(nav), логотип
  2. <nav></nav> - навигация сайта
  3. <footer></footer> - самый конец(подвал) сайта в нем обычно располагают контактные данные, кто создал сайт и др.
  4. <div></div> - блочный контейнер без смысловой нагрузки используется везде где нельзя использовать другие теги
  5. <span></span> - строчный тег без смысловой нагрузки используется для изменения оформления части строки или слова
  6. <p></p> - блочный тег параграф
  7. <h1></h1> - заголовок первого уровня (всего их 6 уровней и 1 должен быть один на страницу)
  8. <title></title> - заголовок страницы помещается в head
  9. <ul></ul> - маркированный список (используется с li)
  10. <ol></ol> - нумерованный список (используется с li)
  11. <li></li> - элемент списка
  12. <b></b>- делает текст жирным
  13. <em></em> - делает текст курсивным
  14. <big></big> - делает текст большим
  15. <small></small> - делает текст маленьким
  16. <br> - делает перенос текста на новую строку (не закрывается)
  17. <a href="путь до ссылки" target="_blank"></a> - создаёт ссылку на сайт или на объект (картинку, пункт навигации и др.)
    target="_blank" - атрибут открытия новой вкладки при переходе по ссылке
  18. <img src="путь к изображению" alt="альтернативный текст"> - тег добавления картинки (не закрывается)
  19. <style></style> - без подключения файла позволяет добавить CSS стили на страницу (лучше не использовать)
  20. <table border="1" width="100%" height="300"></table> - создает границы таблицы
    border="1" - обводка границ таблицы width="100%" - ширина таблицы height="300" - высота таблицы

    Теги для построяния таблиц:
    1. <tr></tr> - строки таблицы
    2. <th></th> - заголовок таблицы
    3. <td></td> - ячейка таблицы

    Пример работы с таблицей:
    <th></th> - заголовок 1 <th></th> - заголовок 2
    <td></td> - ячейка 1 <td></td> - ячейка 2
    <td></td> - ячейка 3 <td></td> - ячейка 4
    <td></td> - ячейка 5 <td></td> - ячейка 6
    <td></td> - ячейка 7 <td></td> - ячейка 8
  21. <form action="#"></form> - создаёт границы формы (например формы обратной связи)
    action="" - атрибут для серверного языка (back-end) для того, чтобы управлять данными полученными с формы

    В формы вставляются слудующие теги:
    1. <input type="text" name="text" placeholder="Введите ваше имя"></input> - создание разных полей ввода
      Имеет слудующие атрибуты:
      1. type="значение" - атрибут типа сообщает браузеру к какому типу относится элемент
        данный атрибут имеет слудующие значения:
        • ="text" - текстовое поле для ввода текста
        • ="password" - текстовое поле для ввода пароля
        • ="tel" - текстовое поле для ввода телефона
        • ="number" - числовое поле
        • ="radio" - радио кнопка (круг с точкой), чтобы работал в паре с другими радио кнопками надо им дать одинаковые имена (name="")
        • ="checkbox" - кнопка чекбокса (квадратик с галочкой), должно быть уникальное имя у каждого чекбокса
        • ="file" - кнопка добавить файл для пользователя
        • ="submit" - кнопка для конечной отправки данных с формы на сервер (используется с value="Отправить")
        • ="reset" - кнопка очистки формы
      2. name ="name" - атрибут имя input нужен серверу для обработки формы
      3. value="значение" - значение по умолчанию в полях
      4. placeholder="значение" - полупрозрачная надпись подсказки для чего преднозначено поле (напирмер "Введите ваше имя")
    2. <label for="id тега input"></label> - устонавливает связь с полем ввода (input) и сообщает, что тект помещённый в данный тег относится именно к данному полю (input)
    3. <select" multiple></select> - создаёт элемент выпадающего списка (работает только c option)
      • multiple - атрибут создаёт сразу открытый список для выбора значения
    4. <option"></option> - элекмент входящий в выподающий список (select)
  22. <textarea name="" id="" cols="30" rows="10" name="textarea"></textarea> - большое текстовое поле
    Имеет слудующие атрибуты:
    1. cols="30" - количество колонок
    2. rows="10" - количество строчек
Какие бывают атрибуты: (универсальные)
  1. style="CSS стили" - атрибут стилей можно вставлять css код
  2. width="значение" - изменить ширину элемента
  3. height="значение" - изменить высоту элемента

2. Метатеги (36 минута 17 секунда)

  • мататеги (meta) - это теги размещённые в head на странице, они используются для индексирования поисковиками, эти теги описывают страницу с технической и визуальной стороны
Какие бывают метатеги:
  1. <meta charset="UTF-8"></meta> - с данным атрибутом указывает к какой кадировке символьной будет относится страница
  2. <meta name="description" content="Это моя первая страница "></meta> - с значением description мы указываем описание страницы в выдаче поисковиков
    Пример (выделено красным):
  3. <meta name="author" content="Andrey"></meta> - Тег который говорит о разработчике страницы
  4. <meta name="keywords" content="сайт визитка, разработка сайтов под ключ"></meta> - тег ключевые слова для поисковых запросов пользователя. (не актуален)
  5. <meta name="robots" content="index, follow"></meta> - мета тег разрешающий роботам поисковых систем индексировать нашу страницу.
    ="noindex" - разрешение роботу индексировать страницу
    ="noindex" - запрет робботу индексировать страницу
    ="follow" - разрешает следовать по ссылкам роботу
    ="nofollow" - запрещает следовать по ссылкам роботу
    ="all" - разрешаем и индексацию и следовать по ссылкам
  6. <meta property="og:title" content="Это моя первая страница"></meta> - описание для социальных сетей для пормирования рекламных постов
